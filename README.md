# Retail Sales Forecasting & Inventory Optimization System

## Overview
This project predicts retail sales and optimizes inventory using machine learning.

## Features
- Sales Forecasting (Random Forest)
- Inventory Optimization (ROP, Safety Stock)
- Visualization Dashboard (Streamlit)

## Tech Stack
Python, Pandas, Scikit-learn, Streamlit

## Run Project

bash
python main.py
streamlit run app/app.py
Output
Forecast Graph
Inventory Table
CSV Output
